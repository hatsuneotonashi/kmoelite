# 行为准则

Kmoe Client 社区应保持尊重、克制和专业。项目是非官方漫画阅读器和个人阅读管理工具，讨论应围绕合规、安全、可维护的源码协作。

## 鼓励的行为

- 清晰描述问题、复现步骤和改动目的。
- 尊重不同平台、语言和使用习惯。
- 对安全、版权、站点条款和账号隐私保持谨慎。
- 用脱敏日志、fixture 和最小复现协作。

## 不接受的行为

- 骚扰、歧视、人身攻击或恶意挑衅。
- 发布账号、密码、Cookie、Session、Token、授权 URL、本机私有路径或真实下载文件。
- 要求或提交绕过登录、会员、配额、验证、反滥用或版权限制的内容。
- 上传未经授权的漫画图片、封面缓存、下载文件或其他受版权保护内容。

## 处理方式

维护者可以编辑、隐藏或关闭违反本准则的 issue、PR、评论或讨论。严重或重复违反者可能被限制参与项目。

安全问题请按 [SECURITY.md](SECURITY.md) 处理，不要在公开 issue 中发布漏洞利用细节或敏感信息。
